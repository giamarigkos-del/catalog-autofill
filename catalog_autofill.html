1
Κατέβασε το Extension

Αποσυμπίεσε το ZIP σε φάκελο
2
Άνοιξε τα Extensions
Πάτα: chrome://extensions
ή: 3 τελείες ⋮ → Extensions → Manage Extensions
3
Developer mode → ON
Toggle πάνω δεξιά
4
"Load unpacked" → επίλεξε φάκελο
Επίλεξε τον φάκελο που αποσυμπίεσες
5
Pin-αρε το extension στη γραμμή
Πάτα το 🧩 εικονίδιο πάνω δεξιά στο Chrome →
βρες το ⚡ efood Catalog Scraper → πάτα το 📌
Το ⚡ θα εμφανιστεί μόνιμα στη γραμμή του Chrome
✓ Μετά την εγκατάσταση: το ⚡ εικονίδιο εμφανίζεται αυτόματα στη γραμμή του Chrome. Δεν χρειάζεται bookmark! Πήγαινε στο κατάστημα (Wolt ή Box) → πάτα ⚡ → "Εκκίνηση εξαγωγής" → περίμενε χωρίς να αγγίξεις τη σελίδα → κατέβασε JSON → ανέβασέ το εδώ
